### Admin Features
- **Announce Projects** — Add films, series, music projects with title, category, status, description, director, year and image. Toggle Live/Draft visibility anytime.
- **Blog & News** — Write and publish blog posts and announcements. Publish or hide posts anytime.
- **Casting Forms** — Create casting calls per project with full role details. Open or close the form anytime with one click.
- **Submissions** — View all casting applications in a table. Export to Excel (.xlsx) with one click — includes all 19 applicant fields.

---

## Casting Form
When a casting form is set to **Open** in the admin portal, visitors can submit a full application including:
Name, DOB, Gender, Age, Mobile, Email, City, Role, Experience, Previous Work, Languages, Instagram/Portfolio, Special Skills, Availability and more.

All submissions are exportable to Excel directly from the admin panel.

## Technical Notes
- **Single file** — entire website is contained in `index.html`
- **No backend required** — runs entirely in the browser
- **Data storage** — uses `localStorage` for admin data (device-specific for demo purposes)
- **Excel export** — powered by SheetJS (loaded from CDN)
- **Fonts** — Cormorant Garamond, Syne, DM Mono (Google Fonts)
- **Fully responsive** — works on mobile, tablet and desktop

---

## Upgrading to a Live Database
Currently data is stored in `localStorage` (browser-only). To make admin changes visible to all visitors, connect to **Firebase Firestore** — the rest of the code stays the same.
